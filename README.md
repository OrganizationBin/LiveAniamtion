# LiveAniamtion

iOS直播动画，特效动画，礼物动画。

做到动态在线配置，不需要依赖版本发布，这个方案目前在直播客户端已使用两年。

# 使用步骤
1.将图片与配置文件打包，上传至服务端，然后客户端下载。获取路径，在送礼物时候，直接调用，礼物动画就出来了。

2.支持2种动画：gif轮播图动画与自定义骨骼动画。

3.主要配置 animation.json 与 animationConfig.plist，然后加入素材图即可。
